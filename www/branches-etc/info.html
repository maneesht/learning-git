Branches and other fun stuff!

WTF is a branch? Do I need a tree?

Nope! You don't need to go buy a tree (for this lesson at least). A branch is basically something that contains multiple commits. You need them to divide up tasks you're working on. For example, you'd have a main branch that everyone works off of and then several other branches branched off for features like adding another screen, implementing login and such.

Let's get started!

Let's start off from the last example and type git branch -b fun-branch

This basically creates a new branch off of your master branch. Basically, it represents that we're starting to work on something. Now go ahead and open up your hello.txt and add something to it. Now, go ahead and create a commit! Done? That's it! You've created a branch with a commit!

What if I want to make it a part of the main branch?

Excellent question! Go ahead and checkout the master branch by typing git checkout master. Now, type git merge fun-branch and you should have your latest commits on the master branch!

Let's get into what we just did.

When you are working independently on something (a feature or whatever), the first protocol is to create a new branch. This is to separate your work from whatever is on master. Now, you must also understand that each branch has a root and typically on projects, the root is a commit on the master branch. Once you're done with your work you need to "squash" it and then get it merged into master.

But I don't like squash...

Good, because I don't either. Usually after you're done with the featuer you're working on, you end up with this long chain of commits on your branch. That's usually not fun to merge, so you need to combine the commits into one. This is called "squashing"

Let's create another commit. Add another line to hello.txt and then commit it. Now, type git rebase -i master

The i is for interactive and master is what you want to rebase onto. We usually want to rebase onto master. Now, you should get some editor to pop up. Now, leave one pick, and replace the rest with s.
