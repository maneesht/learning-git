Making Commits

Getting your environment set up

Well, now that you know what a commit is, let's get started with a project! As far as what you need to get started, it's not much. Go ahead and create a new directory, or find an existing one, and open up a command prompt or terminal. CD into the directory you want git set up, and then type git init. Then you're done with the set up!

Making your first commit.

The parameter add is basically adding something to git. -A means "all"
